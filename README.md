# FAQ

## pkg 出现安装失败？
> 请先检查是否已完全退出微信

安装插件需要完全退出微信

> 已经完全退出了微信，但还是失败？

可能是没安装开发者组件导致的，安装插件需要开发者组件\
`command` + `空格`输入`终端`，回车后，在终端输入 `otool` 回车，激活开发者组件下载

- **会封号吗？**

> 所有处理均为本地处理，不经过任何网络传输，有能力的可以自行抓包检测\
> ⚠️ 多开功能不建议使用，因为本来官方就不支持多开

- **录屏权限未开启？**
> 前往设置中对微信的录屏权限进行打开\
> 若打开后仍不生效，请点击减号删除微信重新添加，重启App即可
